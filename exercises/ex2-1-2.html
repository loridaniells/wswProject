When to Use <section> Tag

The <section> tag: Use to group thematically-related content. It should not be used as a div element as a div is not semantic but more for styling. Always ask yourself, Is all of the content related? Source: http://stackoverflow.com/questions/4781077/html5-best-practices-section-header-aside-article-tags


99% of the time a section element will have an h1 tag. Here is a tool for checking your markup: HTML5 outiner tool This tool will notify you if you have a section that requires a h1 tag.

The section element is not a generic container element. When an element is needed for styling purposes or as a convenience for scripting, authors are encouraged to use the div element instead. Source: http://html5doctor.com/the-section-element/

The W3C states:

Examples of sections would be chapters, the various tabbed pages in a tabbed dialog box, or the numbered sections of a thesis. A Web site's home page could be split into sections for an introduction, news items, and contact information.
